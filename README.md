# Creator Studio

**Six skills for video and podcast creators who publish consistently and grow.** — built in-house by [Skill&nbsp;Me](https://skillme.dev).

Creator Studio gives solo creators a complete writing toolkit for the full content lifecycle, from scripting and hooks through distribution and planning. Every skill is opinionated, format-aware, and built for creators who ship regularly rather than occasionally.

⭐ **If this is useful, star the repo** — it's how we gauge what to build next.

## Install

- **From the catalog:** [skillme.dev/pack/creator-studio](https://skillme.dev/pack/creator-studio) — install the whole pack into Claude in one step.
- **With the skills CLI:** `npx skills add aouellets/creator-studio`
- **Manually:** copy any `skills/<slug>/SKILL.md` into your Claude skills directory.

## Skills in this pack

- **[YouTube Script Writer](skills/youtube-script-writer/SKILL.md)** — Write retention-optimized YouTube scripts with a strong hook, clear structure, and satisfying payoff.
- **[Video Hook Writer](skills/video-hook-writer/SKILL.md)** — Write high-impact opening hooks for short-form video that stop the scroll in the first 3 seconds.
- **[Content Repurposing](skills/content-repurposing/SKILL.md)** — Turn one long-form piece of content into platform-native assets across channels.
- **[Show Notes Writer](skills/show-notes-writer/SKILL.md)** — Write complete podcast show notes with episode summary, timestamped chapters, key takeaways, and guest or resource links.
- **[Thumbnail Concept](skills/thumbnail-concept/SKILL.md)** — Ideate high-CTR thumbnail concepts and paired title options for YouTube videos.
- **[Creator Content Calendar](skills/creator-content-calendar/SKILL.md)** — Plan a sustainable content calendar with defined pillars, posting cadence, and theme rotation.
- **[Tweet Thread Builder](skills/tweet-thread-builder/SKILL.md)** — Viral thread structure: a scroll-stopping hook, one idea per post, and cliffhangers that pull readers down.
- **[LinkedIn Post Writer](skills/linkedin-post-writer/SKILL.md)** — Viral LinkedIn posts: hook formulas, formatting patterns, and CTAs that actually get engagement.

# Creator Studio

**For solo video & podcast creators: script, hook, package, and distribute every episode.** — built in-house by [Skill&nbsp;Me](https://skillme.dev).

Reach for Creator Studio when you run a video or podcast channel alone and the production line keeps stalling — a blank script, a weak hook, a thumbnail that doesn't earn the click, show notes you skip, and a calendar you can't sustain. It carries one episode end to end: write a retention-built YouTube script, open short-form with a scroll-stopping hook, package the video with a high-CTR thumbnail and title, ship complete podcast show notes, repurpose the source into platform-native posts, and lock a cadence you can actually keep. Built for creators who publish every week, not occasionally.

⭐ **If this is useful, star the repo** — it's how we gauge what to build next.

## Install

- **From the catalog:** [skillme.dev/pack/creator-studio](https://skillme.dev/pack/creator-studio) — install the whole pack into Claude in one step.
- **With the skills CLI:** `npx skills add aouellets/creator-studio`
- **Manually:** copy any `skills/<slug>/SKILL.md` into your Claude skills directory.
